# Senior Automated Dashboard v2

Production-grade internal dashboard that automatically syncs data from all 24 senior repositories using GitHub REST API and Supabase.

**Features:**
- Live overall senior progress (currently 78%)
- Repository health scores
- Blog post analytics
- Automated streak counter (learning days)
- Phase completion with Mermaid charts
- Private notes synced across devices

**Tech:** Next.js 15, TypeScript, Tailwind, Recharts, Octokit (GitHub API), Supabase Auth

**Live URL (after deployment):** https://dashboard-v2.moekyawaung.dev
